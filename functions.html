Documentation

<bool> isluau(<void>)

• Returns true if the game is running LuaU

<string> getnamecallmethod(<void>)

• Returns the namecall method if the function is called in an __namecall metatable hook

<table> getgenv(<void>)

• Returns the global environment in a table

<table> getrenv(<void>)

• Returns Roblox's environment in a table

<table> getreg(<void>)

• Returns Roblox's lua registry in a table

<table> getgc(<void>)

• Returns Roblox's gc functions/table in a table

<table> getsenv(<LocalScript, ModuleScript> Script)

• Returns the environment of Script and returns nil if the script is not running

<table> getrawmetatable(<table> value)

• Retrieve the metatable of value regardless of value's metatable's __metatable field. Returns nil if it doesn't exist

<bool> setrawmetatable(<object> o, <table> mt)

• Sets o's metatable to mt even if the __metatable field exists in o's metatable

<table> getflags(<void>)

• Returns loaded fflags in a table

<string> getclipboard(<void>)

• Returns whatever is copied on the clipboard as a string

<string> setclipboard( ... )

• Sets clipboard data to all given arguments

<bool> isreadonly(<table> table)

• Return's table's readonly value

<void> setreadonly(<table> table, <bool> val)

• Sets table's read-only value to val

<bool> checkcaller(<void>)

• Return true if the current thread was not called from game script

Keyboard/Mouse Library

<void> keypress(<number> keycode)

• Simulates a key press for the specified keycode

<void> keyrelease(<number> key)

• Releases key on the keyboard

<void> mouse1click(<void>)

• Simulates a left mouse button click

<void> mouse1press(<void>)

• Simulates a left mouse button press without releasing it

<void> mouse1release(<void>)

• Simulates a left mouse button release

<void> mouse2click(<void>)

• Simulates a full right mouse button click

<void> mouse2press(<void>)

• Clicks down on the right mouse button

<void> mouse2release(<void>)

• Simulates a right mouse button release

<void> mousescroll(<number> px)

• Scrolls the mouse wheel virtually by px pixels

<void> mousemoverel(<number> x, <number> y)

• Moves the mouse cursor relatively to the current mouse position by coordinates x and y

<void> mousemoveabs(<number> x, <number> y)

• Moves the mouse to the x and y coordinates in pixels from the top left of the window
